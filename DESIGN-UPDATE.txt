🎨 DESIGN TOKENS:

Colors:
- Primary: #FF8F00 (amber)
- Hover: #E67E00 (darker amber)
- Focus: #FFF8E7 (amber light)
- Text: #1A1A1A → #4A4A4A → #8A8A8A
- Borders: #E8E4DC / #C8C0B4 (warm)
- Background: #F7F8FA (warm off-white)

Semantic:
- Success/Debit: #16A34A (green)
- Error/Kredit: #DC2626 (red)
- Info: #3B82F6 (blue)

Spacing:
- 8px, 12px, 16px, 20px, 24px (primary scale)

Radius:
- Chip: 5px
- Button: 7px
- Card: 10px
- Modal: 12px

Shadows:
- Card: subtle (0 1px 3px)
- Modal: prominent (0 8px 40px)
- Focus: amber halo (0 0 0 3px)

Typography:
- Font: Inter (400/500/600/700)
- Base size: 13px
- Line height: normal (dense)
- Variant: tabular-nums
